# Monthly Report Generator

This Flutter application allows users to input daily data for an entire month and export it as an Excel file.

## Features
- Input daily data for:
  - Encroachments (in numbers)
  - No. TLP
  - WP
  - ROW Pillars damaged
- Export the complete monthly dataset to an `.xlsx` file.
- Responsive design for mobile and desktop.

## Tech Stack
- Flutter
- `excel` package for generating Excel files.
- `file_saver` package for saving files across platforms.

## Setup and Run
1. Ensure Flutter is installed and configured.
2. Run `flutter pub get` to install dependencies.
3. Run `flutter run` to launch the application.
